Easily transform World of Warcraft card names into links that show the card information in a
tooltip when hovering over then.

== Description ==

The plugin adds a button in the visual post editor. To use it, just select the text that
represents a World of Warcraft card name, and click the button. It will turn the text into a 
link to the card's page on http://deckbox.org You cannot see the mouseover tooltips directly 
in the editor, but when you publish or preview the post, hovering over the link will show the
card's information in a tooltip.

== Installation ==

1. Head over to the "Install Plugins" section of your Admin Panel, and use the "Upload" link
    to install the zip file.
2. Activate the plugin through the 'Plugins' menu.
3. Use the "WoW" button in the editor.

Alternatively, you can search for "WoW TCG" from the Admin panel Plugins section, select this 
plugin and click 'Install'.
